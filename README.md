## 开发工具

1. 配置node.js环境并熟悉node.js
2. 熟悉electron
3. ~~熟悉React~~

## 运行该项目？

    1.同步或克隆最新代码
    2.确保已配置好node.js环境，然后在控制台中切换到工程目录
    3.控制台中至少运行一次：npm install
    4.控制台输入npm start启动程序
