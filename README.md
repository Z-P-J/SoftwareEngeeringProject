# 15组软件工程项目：MusicPlayer

    基于 electron-vue 开发的跨平台音乐播放器

## 开发工具

    VS Code

## 前期准备

1. 配置node.js环境并熟悉node.js
2. 熟悉electron
3. 熟悉Vue
